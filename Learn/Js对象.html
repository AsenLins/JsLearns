对象是Js的基础，函数，数组等是对象的子类型。 ## Js的语言类型（基本类型） - string - number - boolean - null - undefined - object typeof 是用来判断当前的基本类型 ``` var str="hello world"; var num=1; /*输出string*/ console.log(typeof str); /*输出number*/ console.log(typeof num); ``` null类型陷进： typeof返回object,是因为js的对象在二进制前三位都是0，而null默认都是0，所以前3位肯定都是0，会被判定为是对象（object）.这是Js的一个Bug ``` console.log(typeof object); ``` ## 内置对象 - String - Number - Boolean - Function - Object - Array - Date - RegExp - Error instanceof判断类型是属于哪个实例(true or false) ``` var strObj=new String("hello world"); console.log(strObj instanceof String); ``` Js引擎在特定情况会自动把基本类型的字面量转换为对象： ``` var strLength="Asen".length; /*输出4*/ console.log(strLength); var numFix=123.1.toFixed(2); /*输出123.10*/ console.log(numFix); var str="Asen"; /*并不是String的对象*/ console.log(str instanceof String); /*在特定时候会转换为String对象，输出4*/ console.log(str.length); ``` null和undefined只有字面形式，没有构造形式。 Array,Object,Date,RegExp，文字形式还是构造形式，都是对象，不是字面量，用构造形式创建可以添加一些额外的参数。 Error会在抛出异常时自动创建 ## 对象内容的存储方式 属性访问方式 - 属性名称必须是标准的标识符，不能有其他非法字符，如：！这些. ``` var obj={ name:"Asen", age:10 } /*这种就是属性访问*/ console.log(obj.name); ``` 键值对访问： - 键值对访问的键值，可以是任意类型 - 可计算属性 ``` var obj={ name:"Asen", age:20 } var keyname="Asens"; obj["dc!3"]="test"; /*键值对访问*/ console.log(obj["name"]); console.log(obj["dc!3"]); ``` ## 对象的属性和方法（函数） Js的函数是不属于任何一个对象，函数的运行取决于当前运行的上下文对象。 ``` function Func(){ console.log("Asen"); } var FuncObj=Func; var obj={ FuncAttr:Func, FuncName:"Asen" }; /*三个都是引用同一个函数Func*/ console.log(Func); console.log(FuncObj); console.log(obj.FuncAttr); ``` ## 数组对象 - 数组也是一个对象。 - 给数组添加属性的时候，要留意键值对的值名称。 ``` var objArray=["1",2]; /*使用键值对的方式添加attr属性到数组中*/ objArray["attr"]=3; /*若属性名称与数字下标过于类似，会自动转换为objArray[1],这时会改掉数组下标1的值*/ objArray["1"]="3"; ``` ##　对象的属性描述符 - writable(可写) - enumerable（可枚举） - configurable（可配置） 使用Object.defineProperty(..)来添加属性或者定义属性的特性。 ``` var Obj={}; /*可定义属性以及特性*/ Object.defineProperty(Obj,a,{ value:10, writable:true, enumerable:false } ); ``` asd
