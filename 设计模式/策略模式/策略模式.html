## 定义一系列算法，并且把他们封装起来，并使他们可以互相转换。 - 策略模式的优点： - 可消除分支。 - 灵活可变。 - 解耦。 例子： ``` var MoneyComputeObj={ "A":function (money) { return money*1; }, "B":function(money){ return money*2; }, "C":function(money){ return money*3; } } function MoneyCompute(money,level) { return MoneyComputeObj[level](money); } console.log("============计算分级金额==========="); console.log(MoneyCompute(10,"A")); //输出10 console.log(MoneyCompute(10,"B")); //输出20 console.log(MoneyCompute(10,"C")); //输出30 ```
